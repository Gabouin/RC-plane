# RC Plane Project

![License](https://img.shields.io/badge/License-MIT-blue.svg)
![Project](https://img.shields.io/badge/Project-Hardware-informational)

A student-built RC plane made from foamboard, 3D-printed parts, and off-the-shelf electronics.

## Overview

This repository documents the full project lifecycle:
- airframe design
- CAD files
- electronics and wiring
- bill of materials
- build journal

Flight demo: [YouTube short](https://youtube.com/shorts/v7-GCenTMmM)

## Repository structure

| Path | Description |
| --- | --- |
| [`CAD/`](./CAD) | 3D files of the aircraft (`.f3d` and `.step`) |
| [`kicad wiring diagram/`](./kicad%20wiring%20diagram) | KiCad electrical schematic and netlist |
| [`BOM RC plane.csv`](./BOM%20RC%20plane.csv) | Bill of materials and sourcing links |
| [`journal.md`](./journal.md) | Build notes and project progress |
