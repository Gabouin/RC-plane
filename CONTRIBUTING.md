# Contributing

Thanks for your interest in improving this project.

## Before opening a pull request

1. Open an issue (or comment on an existing one) to align on the change.
2. Keep pull requests focused and small.
3. Update documentation when the change affects files, process, or usage.

## Scope for this repository

Helpful contributions include:
- design improvements to CAD files
- clearer wiring/schematic documentation
- BOM corrections and sourcing alternatives
- build documentation and test-flight reporting

## Pull request checklist

- [ ] Changes are clearly described
- [ ] Related docs are updated (`README.md`, `journal.md`, BOM if relevant)
- [ ] File names and paths remain consistent
- [ ] No unrelated files are modified

## Code of conduct

Be respectful, constructive, and collaborative in all discussions.
