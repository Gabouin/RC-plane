# Project Journal

This journal tracks key milestones and decisions for the RC plane build.

## Goals

- Build a lightweight and stable foam-based RC plane
- Integrate reliable control electronics (receiver, ESC, servos, motor)
- Validate airframe and electronics with real test flights

## Milestones

### Phase 1 — Design and Planning
- Defined initial aircraft geometry and dimensions
- Prepared digital structure plan in Canva
- Built the first CAD model for review and iteration

### Phase 2 — Parts and Electronics
- Finalized parts list and sourcing links in the BOM
- Selected powertrain components (motor, ESC, LiPo)
- Prepared servo linkage and wiring approach

### Phase 3 — Build and Integration
- Assembled main airframe components
- Installed electronics and control surfaces
- Connected and checked wiring paths from schematic

### Phase 4 — Flight Testing
- Completed first test flight
- Captured footage for quick validation of stability and control
- Identified next improvements for performance and durability

## Next Iteration Ideas

- Reduce total weight where possible
- Improve landing gear and fuselage reinforcement
- Refine propeller + motor pairing for better thrust efficiency
- Add more structured flight logs (weather, battery, trim settings, outcomes)
